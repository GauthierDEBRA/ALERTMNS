# PSG Fan Zone

Site fan non officiel du Paris Saint-Germain, 100 % statique (HTML/CSS/JS vanilla).
Aucune dépendance, aucun build : ouvre simplement `index.html` dans un navigateur.

## Pages
- `index.html` — Accueil avec hero, actualités et prochains matchs
- `players.html` — Effectif complet avec filtres par poste et fiches joueurs
- `matches.html` — Résultats et prochains matchs avec détail des buts
- `stats.html` — Stats de saison, classement Ligue 1, meilleurs buteurs
- `history.html` — Palmarès, timeline historique et infos Parc des Princes

## Lancer le site
```bash
cd psg-site
python3 -m http.server 8000
# puis ouvrir http://localhost:8000
```
Ou simplement double-cliquer sur `index.html`.
